# Visualization Tools Introduction

Folder ini berisi dokumentasi mengenai pengenalan awal terhadap Power BI Desktop sebagai alat utama dalam proses Business Intelligence, mulai dari instalasi hingga pembuatan visualisasi sederhana.

## Ringkasan Proyek
Tujuan utama dari modul ini adalah memahami ekosistem Power BI dan menguasai penggunaan antarmuka dasarnya untuk mengolah data menjadi laporan visual yang informatif.

## Aktivitas Utama
* **Pengenalan Antarmuka:** Menjelajahi tiga tampilan utama pada Power BI Desktop: Report View (untuk mendesain visualisasi), Data View (untuk melihat dan mengolah tabel), dan Model View (untuk mengatur hubungan antar tabel).
* **Impor Data:** Mempraktikkan proses pengambilan data dari berbagai sumber, seperti file Excel, ke dalam lingkungan Power BI.
* **Pembuatan Visualisasi Dasar:** Belajar membangun grafik sederhana menggunakan panel Visualizations dan mengatur kolom data melalui panel Data.
* **Navigasi Panel:** Memahami fungsi Ribbon untuk akses fitur, serta panel Filters untuk menyaring informasi yang ditampilkan pada laporan.

## Wawasan Utama (Key Insights)
* **Kemudahan Integrasi:** Memahami betapa efisiennya Power BI dalam menghubungkan data mentah dengan elemen visual secara langsung.
* **Fleksibilitas Desain:** Mengidentifikasi berbagai pilihan visualisasi standar yang tersedia untuk berbagai kebutuhan analisis data dasar.
* **Alur Kerja BI:** Mengenali alur kerja standar dalam Power BI, mulai dari persiapan data hingga publikasi laporan sederhana.

## Alat dan Fitur yang Dipelajari
* **Power BI Desktop:** Tools utama untuk pembuatan laporan.
* **Panel Visualizations & Data:** Komponen kunci untuk menyusun elemen grafik.
* **Ribbon & Pane Navigation:** Navigasi fitur-fitur penting dalam aplikasi.
